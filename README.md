Welcome to the ColSocs repository for tracking, auditing, and maintaining the taxonomic checklist of British Coleoptera. 

The goal of this project is to enable the UK Species Inventory to be a reliable, up-to-date ledger of UK beetle species, accommodating new discoveries, dynamic name changes and synonymies

---

## 🏛️ How This Repository Works

To keep the master database highly accurate and free from unverified edits, we use a curated workflow:
*   **Official Issues:** Restricted to maintainers and administrators to track finalized changes and active system tasks.
*   **Community Input:** Open to everyone! All suggestions, corrections, and new records are handled entirely through the **Discussions** tab.

---

## 💬 How to Contribute (Discussions)

If you have spotted a missing species, a recent name change in literature, or a beetle newly recorded in Britain, please navigate to our **Discussions** tab to open a thread. 

We organize community input into specific categories to keep the taxonomy queue clean:

### 🔍 Available Categories
*   **`Issue with taxonomy`** – For reporting spelling errors, outdated family/genus placements, incorrect authorities, or missing synonyms.
*   **`New to Britain`** – For proposing additions to the British checklist based on newly published literature.

---

## 🛠️ Maintainer Workflows

For repository administrators and automated scripts:
> **Note on Issue Creation:** Any community-created issues outside of the Discussions tab are automatically closed, locked, and redirected via our automated GitHub Actions workflow to ensure data consistency. Maintainers with `admin`, `maintain`, or `write` access can still open backend tracking issues natively.

---

## 📜 Community Guidelines

We welcome input from both amateur coleopterists and professional academic entomologists. When posting a new taxonomy discussion, please try to include:
1.  The current scientific name and authority (with year).
2.  The proposed change or addition.
3.  A citation, link, or reference to the published paper or verified record supporting the change.
